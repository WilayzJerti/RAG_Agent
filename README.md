## Описание проекта

RAG Agent использует локальную языковую модель (LLM) `gemma4` через Ollama для генерации ответов на вопросы. Проект позволяет массово обрабатывать таблицы с вопросами и сравнивать полученные ответы с эталонными.

![Language EN](https://github.com/WilayzJerti/RAG_Agent/blob/main/README_en.md)
## Установка и настройка

### 1. Создание виртуального окружения Python

```bash
python -m venv venv
```

### 2. Активация виртуального окружения

**На Windows:**
```bash
venv\Scripts\activate
```

**На macOS/Linux:**
```bash
source venv/bin/activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Установка и настройка Ollama

#### Скачивание Ollama

Скачайте Ollama с официального сайта: https://ollama.ai

#### Установка модели gemma4

После установки Ollama выполните команду:

```bash
ollama run gemma4
```
Она скачает и запустит модель gemma4

## Подготовка файлов данных

Разместите следующие файлы в корневой папке проекта (в одной директории со скриптом `rag_agent.py`):

### 1. `document.txt`
- Скачайте google dock по ссылке которая находиться в описании вашего задания

### 2. `RAG_questions_table.csv`
- CSV таблица с вопросами для обработки
- Должна содержать колонку с вопросами и id (например, ``` row_id,Question,Answer ``` )

Пример структуры:
``` csv
row_id,Question,Answer
11,В каком году Deep Blue победил Каспарова?,100
12,What was AlexNet's error rate in 2012 ImageNet challenge (in percent)?,100
13,Сколько миллионов пользователей ChatGPT достиг за первые пять дней?,100
```

### 3. `RAG_answers.csv`
- CSV таблица с эталонными ответами на вопросы
- Должна содержать колонку с ответами для сравнения и id (например, ```row_id,Answer```)

Пример структуры:
```csv
row_id,Answer
11,1997
12,15.3
13,1
```

## Запуск проекта

### 1. Запустите скрипт RAG Agent

```bash
python rag_agent.py
```

### 2. Ожидание результатов

Скрипт будет обрабатывать вопросы из `RAG_questions_table.csv` по одному. Это может занять некоторое время в зависимости от:
- Количества вопросов
- Размера документации
- Мощности вашего компьютера

### 3. Получение результатов

После завершения работы скрипта будут созданы следующие файлы:

#### `my_answers.csv`
- CSV файл с полученными ответами LLM
- Содержит сопоставление между вопросами и сгенерированными ответами

Пример структуры:
```csv
row_id,Question,Answer
11,В каком году Deep Blue победил Каспарова?,0
12,What was AlexNet's error rate in 2012 ImageNet challenge (in percent)?,15.3
13,Сколько миллионов пользователей ChatGPT достиг за первые пять дней?,0
```
*В данной версии скрипта, в промте прописано чтобы LLM писала 0 если она не знает ответа*
